## 🔍 项目简介

绝大多数语音助手、AI 智能体，都被绑定系统、绑定账号、绑定厂商生态，只能在少数 App 内有限使用，权限巨大、风险极高、容易封号。

VoiceBridge 不走这条路。

我们不接管微信、不接管钉钉、不读取聊天、不介入隐私会话、不触碰账号登录体系。
完全避开风控、避开封禁、避开隐私法律责任。

我们只做好一件事：

### PC 全局 · 轻量语音唤起 + 本地快捷指令 + AI 能力中转

- 全软件通用，不绑定任何厂商、不锁生态
- 本地优先、不上云、不上传隐私、全程可控
- 极低资源占用，旧 Mac  低配 Windows 也流畅
- 可无缝挂载 OpenClaw / 本地大模型 / 桌面智能体

---

## ❌ 为什么我们不做 App 内自动操控？

非常现实、非常稳妥：

1. 微信 / 钉钉 / 飞书 原生已深度接入官方 AI
2. 第三方自动点击、读取聊天极易风控封号
3. 会话隐私敏感，开源项目不能承担泄露责任
4. 大厂原生权限、安全、风控远强于任何第三方

我们不与大厂内卷，我们补齐大厂空白。
